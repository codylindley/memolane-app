Memolane API

The purpose of the Memolane.com Content API is to allow 3rd party sites and developers (from now on referred to as "client applications" or simply "clients") to, in a flexible way, "push" content to Memolane.com from sites that are not directly supported by Memolane.com. Due to the way Memolane.com stores and renders memos, this API provides a great deal of flexibility to client developers with respect to what data is part of the content they push to Memolane and how it is rendered on the site.

A client is only allowed to push content to Memolane.com user accounts where the user has explicitly granted the 3rd party permission to do so. For keeping track of this, Memolane.com uses the OAuth 2 protocol.

The API also allows a clients to read back memos it has previously posted to a user account and to update previously posted content, as well as reading basic user info and statistical information.
